# 推荐的浏览器主题

在之前的版本里面，我为了模仿 Arc 的样式，做了几个渐变的 Vivaldi 主题放在了项目文件里供大家使用。然后我前几天去逛了一下 Vivald Community 的 Themes 专区，发现上面就有很多合适的主题。我就在想，我何必舍近求远自己自制主题，于是我在 Community 尝试了一些热门主题，并挑选了几个我觉得合适的主题提供大家直接使用。使用的方式很简单：
1. 点击主题名字跳转到主题页面
2. 点击主题下方的安装按钮å
